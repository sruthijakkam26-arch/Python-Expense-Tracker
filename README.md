💰 Python Expense Tracker

A simple and user-friendly Expense Tracker built with Python that helps users record, manage, and analyze their daily expenses. This project is ideal for beginners learning Python and basic file handling.

🚀 Features

- Add new expenses
- View all recorded expenses
- Categorize expenses
- Calculate total spending
- Save and load expense data
- Simple command-line interface

🛠️ Technologies Used

- Python 3
- File Handling
- Functions
- Lists and Dictionaries

📂 Project Structure

Expense-Tracker/
│── expense_tracker.py
│── expenses.csv
│── README.md

▶️ How to Run

1. Install Python 3.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run:

python expense_tracker.py
